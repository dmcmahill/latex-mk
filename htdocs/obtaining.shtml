How to Get LaTeX-Mk

Source Format

LaTeX-Mk comes with complete source code which can easily be configured and installed on most Unix-like operating systems with no modifications. In addition, it can be configured and installed easily under WIN32 using the cygwin environment.

LaTeX-Mk source code may be downloaded from the File Release Area of the LaTeX-Mk Sourceforge Project page.

Native Packages

LaTeX-Mk is also available as a native package for the following operating systems. Please note that these are maintained by groups other than the LaTeX-Mk maintainer:

  • NetBSD: LaTeX-Mk Package.
  • FreeBSD: LaTeX-Mk Port.
