Reporting LaTeX-Mk Bugs and Other Feedback

Bug Reports

If you find a bug in LaTeX-Mk or have a suggestion on how to improve LaTeX-Mk, please visit the LaTeX-Mk Sourceforge Project Page. Before filing a bug report, please check the bug database and see if the bug has already been reported. If note, file a bug report there. Remember, your feedback is what helps to improve LaTeX-Mk!
